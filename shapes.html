Shapes and Sizes



How To Use Shapes and Sizes

The Main Page has two buttons which will allow you to choose a between 2D and 3D shapes.


2D and 3D Shapes

Pressing the relevant button will give you access to the section of the app where you can choose from the list of shapes within that section. After you have clicked the shape that you wish to perform calculations for you will be taken to a page where you can enter the required dimension (s). The text below outlines both the possible inputs, and either the required inputs, or in some cases the possible input combinations. As it relates to an Ellipse, and Trapezoid, the value given for the Ellipse perimeter result is as noted on the result page, a very close estimate, this is due to the calculations requires to give a more exact result, as for a Trapezoid, if only Side a,b, and Height are input only a partial result will be displayed. When calculating a triangular prism if the dimensions render a second valid triangular base, two results will be provided and displayed.

It should be noted that all inputs must be of the same unit type, e.g. meters, feet, inches, millimeters, and the results will be displayed in the same unit type. Also where degrees are required for calculation the input is in degrees and not radians, the results will also be in degrees.
